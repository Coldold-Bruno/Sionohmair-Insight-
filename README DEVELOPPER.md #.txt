# Documentation Développeur - Sionohmair Insight

Bienvenue dans le projet Sionohmair Insight.

## Structure du projet

- `/frontend` : Application mobile React Native  
- `/backend` : API backend Node.js / Express  
- `/docs` : Documentation technique  
- `.env.example` : Exemple de variables d’environnement  
- `.gitignore` : Fichiers à ignorer par Git  
- `start-dev.sh` : Script pour lancer simultanément frontend et backend (Linux/macOS)

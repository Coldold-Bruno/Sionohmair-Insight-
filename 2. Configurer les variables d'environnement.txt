2. Configurer les variables d’environnement



Duplique .env.example en .env dans le dossier backend, et remplis les valeurs (ex. JWT_SECRET, DATABASE_URL, etc.).

3. Lancer l’application



Pour le backend (mode dev avec nodemon) :


npm run dev

Pour le frontend (React Native) :


npx react-native run-android  # ou run-ios

Ou via le script :

./start-dev.sh
